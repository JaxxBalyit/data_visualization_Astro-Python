# Environmental Data Insights

**Visualización interactiva de datos ambientales en México** usando **Astro + Tailwind + Plotly.js**.

![Preview](docs/screenshot.png)

## Características
- Gráficos interactivos
- Datos reales (CSV)
- Diseño responsive
- Carga ultrarrápida

## Tecnologías
- [Astro](https://astro.build)
- [Tailwind CSS](https://tailwindcss.com)
- [Plotly.js](https://plotly.com/javascript/)

## Instalación
```bash
git clone https://github.com/JaxxBalyit/environmental-insights.git
```
Posteriormente, ejecutar:
```bash
cd environmental-insights
npm install # o pnpm install
npm run dev # o pnpm dev
```
